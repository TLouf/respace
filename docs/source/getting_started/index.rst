Getting started
----------------

Here you'll find all you need to get yourself started quickly with ReSpace. Although the
library depends heavily on :mod:`xarray`, no prior knowledge of the library is expected. If
things aren't much clearer to you after reading the basics guide, or any
other guide, feel free to raise an issue and tell us how things could be improved. TODO
